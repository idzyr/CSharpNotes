# Debug

## 单步调试

按F11启动程序进入单步调试模式。

可以从监视器中输入要监视的内容查看其值

## 断点调试

在有可能会出现，错误的代码左侧的灰色部分添加一个断点，然后启动程序，程序运行到此断点会自动停止然后在按F11进行单步调试。
