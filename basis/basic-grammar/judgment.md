# 判断

```C#
if(布尔表达式){
    ...
}

if(布尔表达式){
    ...
}else{
    ...
}

if(布尔表达式){
    ...
}else if(布尔表达式){
    ...
}else{
    ...
}
// 定制判断，支持多case匹配一个。
switch(变量或者表达式的值)
{
    case 值1：
        要执行的代码;
        break;
    case 值2:
    case 值2a:    
        要执行的代码;
        break;

    .....
    default: 
        要执行的代码;
        break;
}
```
